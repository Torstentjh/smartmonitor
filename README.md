# SmartMonitor

<div align='center' ><font size='70'>家庭环境监控App</font></div>
&emsp;&emsp;**本项目是将毕业设计进行简单重构，软件使用ReactNative开发，本仓库没有连接Mqtt服务端，仅通过Ws同步多设备的开关状态，也不提供硬件部分代码，只RN的代码以供参考，RN新手，有问题欢迎建议**

## Tech Stack

**Front End**

- Core: React Native, based on React Native CLI
- UI: Antd Mobile RN
- Router: React Navigation
- API Request: Axios
- API State Management: React Query
- Global State Management: Zustand
- Style Framework: Tailwindcss

**Back End**

- Core: Koa.js
- Database: Mysql
